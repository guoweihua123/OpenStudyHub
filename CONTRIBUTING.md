# 贡献指南（CONTRIBUTING）

感谢你对 OpenStudyHub 项目的关注与支持！

本项目是一个面向大学生的开源学习资源与实用工具集合项目，
欢迎通过合理、规范的方式参与项目贡献。

---

## 如何参与贡献

你可以通过以下方式参与本项目：

- 提交 Issue：反馈问题、提出建议或改进意见
- 提交 Pull Request（PR）：完善文档或新增功能

---

## 提交 Issue 规范

在提交 Issue 前，请注意：

1. 确认该问题尚未被提出
2. Issue 标题应简要描述问题
3. 内容应尽量说明问题背景或改进建议

---

## 提交 Pull Request 流程

1. Fork 本仓库到你的个人账号
2. 在本地新建分支进行修改
3. 提交修改并推送到远程仓库
4. 在 GitHub 上发起 Pull Request
5. 等待项目维护者审核并合并

---

## 贡献注意事项

- 提交内容应与项目主题相关
- 保持文档与代码的清晰性和可读性
- 遵循基本的 Git 提交规范

---

## 行为规范

参与本项目的所有贡献者应保持良好的交流态度，
共同维护友好、开放的开源协作环境。
